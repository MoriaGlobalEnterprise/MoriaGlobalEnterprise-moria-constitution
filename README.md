# Constituição Moriá

Este repositório reúne os princípios fundacionais, epistemológicos, institucionais, culturais e de engenharia da **Moriá Global Enterprise**.

A Constituição Moriá não é documentação de um produto específico. Ela está acima dos produtos, projetos e tecnologias da organização e serve como referência para orientar decisões, arquitetura, governança, cultura, desenvolvimento e expansão.
