# RFC na Constituição da Moriá

A Moriá adota RFC como princípio institucional de evolução do conhecimento.

## Três ressignificações complementares

### RFC — Request for Convergence
Toda proposta busca convergência entre evidências, especialistas e realidade observável.

### RFC — Request for Canonization
Após validação, uma proposta pode ser incorporada ao Cânone da Moriá e passar a orientar arquitetura, produtos e processos.

### RFC — Request for Confidence
O objetivo de cada RFC é elevar o nível de confiança de uma hipótese antes de sua institucionalização.

## Fluxo
Observação → Hipótese → RFC → Fricção por evidências → Validação → Cânone → Implementação → Monitoramento → Nova RFC, quando necessário.

Nenhum princípio é imune à revisão diante de evidências superiores.
